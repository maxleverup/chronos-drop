## ✨ Features

### 📤 Send
- **Single file mode** — pick any file, upload it, and get an 8-character access code instantly
- **Multi-file mode** — bundle multiple files into one transfer package under a single code
- **Share sheet support** — select files in your file manager → Share → Chronos Drop, then choose Single or Multi mode from a dialog

### 📥 Receive
- **Receive by code** — enter the 8-character access code to fetch files immediately
- **Single file download** — opens directly with your preferred app or saves to device
- **Multi-file package** — numbered file list with individual open/save buttons for each file

### 🗂️ History
- **Full transfer log** — every send and receive is saved locally in SQLite
- **Session grouping** — transfers are grouped by access code and time (minute-accurate), so repeated use of the same code creates separate history entries
- **Clear individual entries** — delete any history card with one tap, or wipe all with Clear All

### 🔒 Privacy
- **No account required** — zero sign-in, zero profile, zero personal data collected
- **Anonymous codes** — access codes are registered anonymously via Firebase; no identity is attached
- **Local history only** — your transfer history lives only on your device
